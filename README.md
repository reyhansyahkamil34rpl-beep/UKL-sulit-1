# UKL-sulit-1
Deskripsi Program
Program ini digunakan untuk menghitung rata-rata nilai siswa dari sejumlah data nilai yang dimasukkan oleh pengguna.
Jumlah siswa ditentukan oleh pengguna, kemudian pengguna memasukkan nilai masing-masing siswa satu per satu.
Program juga dilengkapi fitur validasi input agar pengguna tidak memasukkan data yang salah (contoh: huruf atau simbol).
Fitur Program
 Input jumlah siswa
 Validasi input jumlah siswa (harus bilangan bulat positif)
 Input nilai setiap siswa dalam bentuk angka (double)
 Validasi input nilai
 Opsional: Jika nilai < 0 maka nilai dianggap 0
 Menghitung rata-rata nilai
 Hasil ditampilkan dengan dua angka di belakang koma
Konsep dan Teknologi yang Digunakan
Konsep/Library	Fungsi
Scanner	Membaca input dari keyboard
Perulangan for	Untuk memasukkan nilai sesuai jumlah siswa
Struktur pengkondisian	Validasi input
DecimalFormat	Mengatur format output desimal
Tipe data double	Menyimpan nilai numerik pecahan
Cara Menjalankan Program
Compile program:
javac UKLsulit1.java
Jalankan program:
java UKLsulit1
Masukkan jumlah siswa dan nilai-nilai mereka sesuai instruksi
Contoh Output
Masukkan jumlah siswa: 4
Nilai siswa ke-1: 80
Nilai siswa ke-2: 90.5
Nilai siswa ke-3: 70
Nilai siswa ke-4: 88
Rata-rata nilai: 82.13
Jika input salah:
Masukkan jumlah siswa: abc
Input tidak valid
Kesimpulan
Program ini membantu pengguna untuk:
Menghitung nilai rata-rata secara cepat dan tepat
Memahami penerapan perulangan, validasi, dan format angka pada Java
Melatih penanganan error dalam input data
